# Support

This is a focused tool with a narrow scope. It is not a general-purpose network
plugin and does not aim to be.

**Bug reports** with reproduction steps are welcome.
**Experience reports** — friction, unexpected behavior, surprising edge cases —
are useful even without a proposed fix.
**Pull requests** may be reimplemented rather than merged as-is to keep the
codebase coherent. Open an issue first if the change is non-trivial.
**Feature requests** outside the stated scope will be declined; forks are
encouraged.

Response time is best-effort. This is not a supported product with an SLA.

For security issues, see [SECURITY.md](SECURITY.md).
